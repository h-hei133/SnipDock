# SnipDock v0.4.0 Verification Checklist

本清单用于 v0.4.0 Markdown 预览验收。

## 自动检查

- [ ] `dotnet build SnipDock.sln` 通过。
- [ ] `dotnet test SnipDock.sln` 通过。
- [ ] Markdown 预览测试覆盖标题、代码块、列表和空内容。
- [ ] Git 状态不包含 `bin/`、`obj/`、`publish/`、`dist/`、`logs/`、`backups/` 或真实数据文件。

## Markdown 预览验证

- [ ] 详情页可以显示 Markdown 预览。
- [ ] 详情页可以切换预览 / 原文。
- [ ] 编辑页可以切换编辑 / 预览。
- [ ] 切换预览不会修改原文。
- [ ] 代码块显示为等宽字体。
- [ ] 普通纯文本仍可正常显示。
- [ ] 复制仍然复制原始文本。
- [ ] 保存仍然保存原始 Markdown 文本。

## 核心回归

- [ ] 新建、编辑、删除正常。
- [ ] 模板变量复制流程正常。
- [ ] 导入 / 导出 JSON 正常。
- [ ] 自动备份和备份恢复正常。
- [ ] Light / Dark 主题下预览可读。
