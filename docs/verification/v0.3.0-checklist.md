# SnipDock v0.3.0 Verification Checklist

本清单用于 v0.3.0 模板变量验收。

## 自动检查

- [ ] `dotnet build SnipDock.sln` 通过。
- [ ] `dotnet test SnipDock.sln` 通过。
- [ ] 变量解析测试覆盖重复变量、无效变量、空变量、缺失变量。
- [ ] Git 状态不包含 `bin/`、`obj/`、`publish/`、`dist/`、`logs/`、`backups/` 或真实数据文件。

## 模板变量验证

- [ ] 内容不包含变量时，复制行为不变。
- [ ] 内容包含 `{{name}}` 时，复制前弹出变量填写窗口。
- [ ] 内容包含多个变量时，每个变量只显示一次。
- [ ] 重复变量填写一次即可在预览中多处替换。
- [ ] 空变量值可以生成最终文本。
- [ ] 取消变量填写不会复制内容。
- [ ] 生成文本可以复制到剪贴板。
- [ ] Prompt、命令、代码片段、笔记类型均可使用模板变量。
- [ ] 命令类型只复制生成文本，不执行。

## 隐私验证

- [ ] 日志不记录用户填写的变量值。
- [ ] 日志不记录生成后的正文。
- [ ] 日志不记录剪贴板内容。

## 核心回归

- [ ] 搜索仍然只搜索标题和标签。
- [ ] 新建、编辑、删除正常。
- [ ] 导入 / 导出 JSON 正常。
- [ ] 自动备份和备份恢复正常。
- [ ] Light / Dark 主题下变量填写窗口可读。
