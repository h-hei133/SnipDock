# SnipDock v0.5.0 Verification Checklist

本清单用于 v0.5.0 批量管理与重复检测验收。

## 自动检查

- [ ] `dotnet build SnipDock.sln` 通过。
- [ ] `dotnet test SnipDock.sln` 通过。
- [ ] 批量删除和批量更新测试通过。
- [ ] 重复检测测试覆盖标题和内容 hash。
- [ ] 导入时疑似重复跳过测试通过。
- [ ] Git 状态不包含 `bin/`、`obj/`、`publish/`、`dist/`、`logs/`、`backups/` 或真实数据文件。

## 批量管理验证

- [ ] 左侧列表支持多选。
- [ ] 多选两个以上条目时显示批量操作栏。
- [ ] 批量删除前有确认。
- [ ] 批量添加标签后动态标签刷新。
- [ ] 批量修改类型后列表刷新。
- [ ] 批量收藏 / 置顶后排序稳定。
- [ ] 批量操作后右侧详情状态稳定。

## 重复检测验证

- [ ] 可按标题检测疑似重复。
- [ ] 可按内容 hash 检测疑似重复。
- [ ] 扫描重复项不会自动删除或合并数据。
- [ ] 导入时疑似重复项会跳过。
- [ ] 普通导入流程仍可新增非重复条目。

## 核心回归

- [ ] 搜索仍然只搜索标题和标签。
- [ ] 命令条目只复制，不执行。
- [ ] 模板变量复制流程正常。
- [ ] Markdown 预览正常。
- [ ] 自动备份和备份恢复正常。
