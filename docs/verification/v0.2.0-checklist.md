# SnipDock v0.2.0 Verification Checklist

本清单用于 v0.2.0 安装与发布体验验收。

## 自动检查

- [ ] `dotnet build SnipDock.sln` 通过。
- [ ] `dotnet test SnipDock.sln` 通过。
- [ ] `dotnet publish src/SnipDock.App/SnipDock.App.csproj -c Release -r win-x64 --self-contained false -o .\publish\SnipDock` 通过。
- [ ] `ISCC.exe .\installer\SnipDock.iss` 通过。
- [ ] Git 状态不包含 `bin/`、`obj/`、`publish/`、`dist/`、`logs/`、`backups/` 或真实数据文件。

## 安装验证

- [ ] 安装包可以在干净 Windows 环境中运行。
- [ ] 安装后可以从开始菜单启动 SnipDock。
- [ ] 勾选桌面快捷方式后，桌面快捷方式可用。
- [ ] 关于页面显示 `v0.2.0`。
- [ ] 关于页面“检查更新”打开 GitHub Releases 页面。
- [ ] 安装后开机自启仍可正常开启和关闭。
- [ ] 自启注册表项使用 HKCU，不需要管理员权限。
- [ ] 自启项路径包含双引号和 `--startup` 参数。

## 卸载验证

- [ ] 卸载程序可以正常完成。
- [ ] 卸载后开始菜单快捷方式被移除。
- [ ] 卸载后桌面快捷方式被移除。
- [ ] 卸载不会删除用户选择的数据目录。
- [ ] 卸载不会删除 `prompts.json`、`settings.json`、`backups\`、`logs\`。

## 核心回归

- [ ] 首次启动配置数据目录正常。
- [ ] 托盘常驻正常。
- [ ] 悬浮球显示和拖动正常。
- [ ] 全局快捷键打开 / 隐藏面板正常。
- [ ] 搜索仍然只搜索标题和标签。
- [ ] 复制条目正常，命令条目只复制不执行。
- [ ] 新建、编辑、删除正常。
- [ ] 导入 / 导出 JSON 正常。
- [ ] 自动备份和备份恢复正常。
