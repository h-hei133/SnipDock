# SnipDock v0.3.0

## 版本定位

`v0.3.0` 将 SnipDock 从静态片段复制器扩展为支持模板变量的片段生成器。

SnipDock 仍然保持 local-first：数据保存在本地，不上传云端，不自动执行命令。

## 主要功能

- 支持在条目内容中识别模板变量，例如 `{{name}}`、`{{content}}`、`{{style}}`。
- 复制包含变量的条目时，会弹出变量填写窗口。
- 用户填写变量后，SnipDock 会生成最终文本并复制到剪贴板。
- 支持 Prompt、命令、代码片段、笔记等所有条目类型。
- 命令条目仍然只复制，不执行。
- 变量填写窗口提供最终文本预览。
- 无变量条目的复制行为保持不变。

## 数据与隐私

- 变量值只用于本次生成和复制。
- 日志不记录用户填写的变量值。
- 日志不记录条目正文和剪贴板内容。
- 不改变 `prompts.json` 数据结构。
- 不强制移动用户数据目录。

## 运行要求

- Windows
- .NET 9 Desktop Runtime

## 下载和运行方式

### 安装包

下载并运行：

```text
SnipDock-Setup-v0.3.0.exe
```

### Zip / 便携包

解压后运行：

```text
SnipDock.App.exe
```

## 已知限制

- 变量名当前支持英文字母、数字和下划线，且不能以数字开头。
- 暂不支持复杂表达式、默认值、条件语句或脚本执行。
- 暂不支持云端 AI 生成。
- 命令条目不会执行，只会复制生成后的文本。

## 验证摘要

- 无变量条目复制行为不变。
- 有变量条目可以填写并复制最终内容。
- 变量解析支持重复变量去重。
- 空变量值允许生成。
- build/test 通过后可发布。
