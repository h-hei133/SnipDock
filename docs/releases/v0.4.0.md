# SnipDock v0.4.0

## 版本定位

`v0.4.0` 聚焦 Markdown 阅读体验，提升 Prompt、代码片段和笔记的浏览可读性。

SnipDock 仍然保持 local-first：数据保存在本地，不上传云端，不自动执行命令。

## 主要功能

- 详情页支持 Markdown 预览。
- 详情页可以在预览和原文之间切换。
- 编辑页可以在编辑原文和预览之间切换。
- 代码块使用等宽字体和独立背景显示。
- 支持标题、列表、引用、分隔线、代码块和普通段落。
- 复制功能仍然复制原始文本。
- 保存流程仍然保存原始 Markdown 文本。

## 数据与隐私

- 不改变 `prompts.json` 数据结构。
- 不改变导入 / 导出 JSON 格式。
- 不记录条目正文、预览内容或剪贴板内容到日志。
- 命令条目仍然只复制，不执行。

## 运行要求

- Windows
- .NET 9 Desktop Runtime

## 下载和运行方式

### 安装包

下载并运行：

```text
SnipDock-Setup-v0.4.0.exe
```

### Zip / 便携包

解压后运行：

```text
SnipDock.App.exe
```

## 已知限制

- Markdown 预览为轻量实现，不包含复杂插件系统。
- 暂不支持在线图片上传。
- 暂不支持富文本编辑器。
- 预览用于阅读，复制仍复制原文。

## 验证摘要

- Markdown 内容可预览。
- 编辑 / 预览切换不改变原文。
- 代码块可读性提升。
- Light / Dark 主题下使用应用主题资源。
- build/test 通过后可发布。
