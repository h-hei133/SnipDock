# SnipDock v0.5.0

## 版本定位

`v0.5.0` 聚焦大量条目下的整理能力，加入批量管理与重复检测。

SnipDock 仍然保持 local-first：数据保存在本地，不上传云端，不自动执行命令。

## 主要功能

- 左侧条目列表支持多选。
- 多选后显示批量操作栏。
- 支持批量删除。
- 支持批量添加标签。
- 支持批量修改条目类型。
- 支持批量收藏。
- 支持批量置顶。
- 支持按标题检测疑似重复。
- 支持按内容 SHA256 hash 检测疑似重复。
- 导入 JSON 时会跳过疑似重复标题或重复内容。

## 数据安全

- 批量删除前会进行确认。
- 批量操作通过服务层一次保存，避免多次中间状态写入。
- 重复检测只提示，不自动删除或合并。
- 导入前仍会创建安全备份。
- 不改变 `prompts.json` 文件名。
- 不改变用户数据目录。

## 运行要求

- Windows
- .NET 9 Desktop Runtime

## 下载和运行方式

### 安装包

下载并运行：

```text
SnipDock-Setup-v0.5.0.exe
```

### Zip / 便携包

解压后运行：

```text
SnipDock.App.exe
```

## 已知限制

- 重复检测是轻量策略，不引入数据库或全文搜索引擎。
- 内容重复检测基于完整正文 hash，不做语义相似度判断。
- 导入遇到疑似重复时默认跳过，不提供复杂合并 UI。
- 批量删除需要谨慎确认。

## 验证摘要

- 批量操作前有确认。
- 批量删除不会绕过服务层。
- 批量操作后列表刷新。
- 重复检测覆盖标题和内容 hash。
- 导入重复检测不阻塞普通导入。
- build/test 通过后可发布。
