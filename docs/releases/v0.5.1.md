# SnipDock v0.5.1

## 版本定位

`v0.5.1` 是一次 UI polish 版本，重点是让 SnipDock 的界面、主题资源、图标接入和阅读体验更适合公开发布与长期维护。

本版本不添加新的业务功能，不改变数据结构，不改名 `prompts.json`，不移动用户选择的数据目录。

## 更新内容

- 收口 Light / Dark 主题资源，补充 surface、focus、success、warning、danger 和 Markdown 相关颜色 token。
- 统一按钮、图标按钮、危险按钮、Dialog、Toast、卡片、Badge、输入框、下拉框、Tab 和滚动条样式。
- 优化主面板、设置页、详情页、编辑页、模板变量窗口和确认窗口的视觉层级。
- 优化悬浮球和右键菜单视觉，不改变拖动、点击、右键菜单和边界限制逻辑。
- 优化 Markdown 预览的标题、段落、引用、代码块、行内代码和列表间距。
- 接入 `logo.png`、`app.ico`、`tray.ico`，并同步应用图标和安装包图标配置。
- 调整全局 UI 字体为更适合中文 Windows 的 `Microsoft YaHei UI, Segoe UI`，代码块仍使用 `Consolas`。

## 下载建议

- 推荐普通用户下载并运行 `SnipDock-Setup-v0.5.1.exe`。
- 如果使用 Zip / 便携包，请解压后运行 `SnipDock.App.exe`。
- 当前发布包仍是 framework-dependent，如无法启动，请先安装 .NET 9 Desktop Runtime。

## 运行要求

- Windows
- .NET 9 Desktop Runtime

## 数据与隐私说明

- 数据只保存在本地用户选择的数据目录。
- 不上传云端。
- 命令类条目只复制，不自动执行。
- 日志不应记录条目正文、模板变量填写值或剪贴板内容。
- 本版本不改变 `prompts.json` 文件名。
- 本版本不移动用户数据目录。
- 本版本继续保留旧 旧版应用 配置兼容迁移逻辑。

## 已知限制

- 暂不提供自动更新器，关于页会打开 GitHub Releases 供手动检查更新。
- 搜索范围仍为标题和标签，不搜索正文内容。
- 安装包不内置 .NET Desktop Runtime。
