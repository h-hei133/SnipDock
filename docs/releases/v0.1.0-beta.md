# SnipDock v0.1.0-beta

## 版本定位

`v0.1.0-beta` 是 SnipDock 的首个 beta 版本，用于公开试用、收集反馈和验证发布流程。

SnipDock 是一个 Windows 本地轻量片段管理器，用于快速保存、搜索和复制 Prompt、命令、代码片段、笔记和常用信息。

## 主要功能

- Prompt / 命令 / 代码片段 / 笔记多类型条目
- 标题和标签搜索
- 类型筛选、标签筛选
- 收藏、置顶、最近使用和使用次数
- 全局快捷键、系统托盘、Windows 悬浮球
- Light / Dark 主题和多 Accent 配色
- 新建、编辑、删除、复制
- JSON 导入 / 导出
- 自动备份、手动备份和备份恢复
- 开机自启
- 从旧 PromptShelf 配置兼容迁移

命令类条目只复制到剪贴板，SnipDock 不会自动执行命令。

## 运行要求

- Windows
- framework-dependent 发布包需要安装 .NET 9 Desktop Runtime

## 下载和运行方式

下载发布包后运行：

```text
SnipDock.App.exe
```

如果无法启动，请先确认已安装 .NET 9 Desktop Runtime。

## 从 PromptShelf 迁移说明

SnipDock 会兼容旧版本 PromptShelf 的 bootstrap 配置：

- 读取旧 `%APPDATA%\PromptShelf\bootstrap.json`
- 迁移到 `%APPDATA%\SnipDock\bootstrap.json`
- 不删除旧配置
- 不移动用户数据目录
- 主数据文件名 `prompts.json` 暂时保留，以保证兼容

## 隐私和本地数据说明

- SnipDock 是 local-first 工具
- 数据保存在用户选择的本地目录
- 不上传云端
- 不自动执行命令
- 命令条目只复制，不运行
- 日志不应记录条目正文和剪贴板内容

主要数据文件：

- `%APPDATA%\SnipDock\bootstrap.json`
- `%LOCALAPPDATA%\SnipDock\logs\`
- 用户选择目录下的 `prompts.json`
- 用户选择目录下的 `settings.json`
- 用户选择目录下的 `backups\`
- 用户选择目录下的 `logs\`

## 已知限制

- 仅支持 Windows
- 暂不提供云同步
- 暂不提供安装器和自动更新
- 搜索只覆盖标题和标签，不搜索正文
- beta 阶段仍需要更多环境验证
